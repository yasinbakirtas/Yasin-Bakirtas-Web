# 🌐 Kişisel Web Portfolyosu

Bu proje, Web Teknolojileri dersi kapsamında, HTML5 ve CSS3 temellerini öğrenirken **hazırladığım ilk web projemdir.** Kodlama dünyasına attığım ilk adımları ve temel web tasarım becerilerimi temsil etmektedir.

## 🛠️ Kullanılan Teknolojiler
* **HTML5:** Yapısal içerik oluşturma.
* **CSS3:** Görsel düzenleme ve temel stil kuralları.

## 💡 Projenin Özellikleri
* **İlk Deneyim:** Web standartlarına giriş aşamasında geliştirdiğim başlangıç seviyesi bir çalışma.
* **Çok Sayfalı Yapı:** Özgeçmiş, hobiler, memleket ve arkadaş listesi gibi farklı bölümlerden oluşmaktadır.
* **Öğrenme Süreci:** CSS ile temel yerleşim düzeni (layout) ve arayüz (UI) tasarımı pratikleri üzerine odaklanılmıştır.
